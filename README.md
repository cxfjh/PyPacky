# PyPacky - Python 打包工具

## 简介

PyPacky 是一个简单的 Python 脚本打包工具，基于 Nuitka 封装，提供图形化界面，方便用户将 Python 脚本打包为可执行文件。

## 安装与运行

### 依赖

- Python 3.6 或更高版本
- Nuitka

### 安装步骤

1. 克隆本仓库到本地：

   ```bash
   git clone https://github.com/cxfjh/PyPacky.git
   ```

2. 进入项目目录：

   ```bash
   cd PyPacky
   ```

3. 安装依赖：

   ```bash
   pip install -r requirements.txt
   ```

4. 运行程序：

   ```bash
   python main.py
   ```
